키워드만 잘 잡아도 내용의 집중력을 향상 시킬수 가 있습니다.  키워드를 잡으려고 노력했습니다.  

3, 가독성과 심미성 확보 
글이 많아도 잘 읽을수 있는게 있는 반면  글이 없어도 글이 잘 않보이는게 있습니다.  글씨, 배경, 위치 등 보기에도 편하고 내용 전달이 좋게 하려고 했습니다.
